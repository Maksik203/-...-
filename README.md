# 🚗 Автомойка «Блеск» — информационная система для управления автомойкой

[![Database](https://img.shields.io/badge/Database-SQLite-green.svg)](https://www.sqlite.org/)

## 📌 О проекте

**Автомойка «Блеск»** — это desktop-приложение для автоматизации работы автомойки. Система предназначена для учёта клиентов, услуг, сотрудников и заказов. Программа разработана на платформе .NET 8.0 с использованием базы данных SQLite.

Проект создан в рамках курсовой работы на тему «Сопровождение информационной системы автомойки».

---

## 🎯 Основные возможности

| Модуль | Описание |
|--------|----------|
| 👥 **Клиенты** | Добавление, редактирование, удаление и просмотр клиентов |
| 🔧 **Услуги** | Управление услугами (цена, время выполнения, описание) |
| 👨‍💼 **Сотрудники** | Учёт персонала (должность, телефон, дата приёма) |
| 📋 **Заказы** | Создание и отслеживание заказов, привязка к клиенту, услуге и сотруднику |
| 🔐 **Авторизация** | Вход по логину и паролю, регистрация новых пользователей |
| 📊 **Панель администратора** | Полный доступ ко всем разделам системы |
