# Сопровождение информационной системы гостиничного бизнеса

Приложение **C# Windows Forms** для Visual Studio. База данных — файл **SQLite** (`Hotel.db`), **ничего дополнительно устанавливать не нужно**.

## Запуск

1. Откройте **`HotelIS.sln`** в Visual Studio 2022.
2. Нажмите **F5**.

Или из командной строки:

```powershell
cd "HotelIS"
dotnet run
```

База создаётся автоматически при первом запуске в папке  
`bin\Debug\net8.0-windows\Database\Hotel.db`.

## Таблицы

- **Клиенты** — ID, фамилия, имя, телефон, паспорт  
- **Номера** — ID, номер, категория, цена, статус  
- **Бронирование** — ID, клиент, номер, даты, итог  

## Требования

- Windows 10/11  
- Visual Studio 2022 с рабочей нагрузкой «Разработка классических приложений .NET»  
- .NET 8 SDK  

Драйвер Microsoft Access **не требуется**.
